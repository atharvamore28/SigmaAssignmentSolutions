Common text formatting tags in HTML

  1. <b>
    • Use the <b> for cases like keywords in a summary, product names in a review.
    • Do not confuse the <b> element with the <strong>.
  2. <i>
    • This tag is generally used to display the technical term, phrase, the important word in a different language.
    • Do not confuse the <i> element with the <em>.
  3. <s>
    • The <s> tag specifies text that is no longer correct, accurate or relevant.
    • The <s> tag should not be used to define deleted text in a document.
  4. <mark>
    • The <mark> tag in HTML is used to define the marked text
    • The <mark> tag denotes relevance.
  5. <u>
    • The <u> tag in HTML stands for underline, and it's used to underline the text enclosed within the <u> tag.
    • If you want to point out spelling errors within the text, then you can use the <u> tag.
